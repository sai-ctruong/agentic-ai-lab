# Day 1 - Character AI Chatbot with Gemini API

This notebook demonstrates the first step of building an AI chatbot in Python with the Google Gemini API. It focuses on loading secrets safely, configuring the Gemini SDK, sending a prompt to a Gemini model, and displaying the response inside a Jupyter notebook.

## What This Project Covers

- Loading `GEMINI_API_KEY` from a root-level `.env` file
- Configuring the `google-generativeai` Python SDK
- Creating a Gemini model client with `gemini-2.5-flash`
- Sending a user message to the model with `generate_content`
- Printing the model response in a notebook
- Establishing the basic pattern used in later days of this repository

## Requirements

- Python 3.12 or newer
- Jupyter Notebook, JupyterLab, or the VS Code notebook interface
- A Google Gemini API key
- A virtual environment, preferably the repository-level `.venv`

## Installation

From the repository root, activate the virtual environment:

```powershell
.\.venv\Scripts\Activate.ps1
```

Install the required packages:

```powershell
pip install -r "Day1 Develop a Character AI Chatbot using GeminiAI API/requirements.txt"
```

If you use `uv`, you can install into the existing `.venv` with:

```powershell
uv pip install --python .\.venv\Scripts\python.exe -r "Day1 Develop a Character AI Chatbot using GeminiAI API/requirements.txt"
```

## Environment Variables

Create or update a `.env` file in the repository root:

```env
GEMINI_API_KEY=your_gemini_api_key_here
```

Do not commit `.env` to GitHub. It contains private credentials.

## How to Run

1. Open `chat-with-geminiAI.ipynb`.
2. Select the Python kernel that points to this repository's `.venv`.
3. Run the cells from top to bottom.
4. Change the value of `my_message` to test different chatbot prompts.

## Main Code Flow

```text
.env
  -> load_dotenv()
  -> os.getenv("GEMINI_API_KEY")
  -> genai.configure(...)
  -> genai.GenerativeModel("gemini-2.5-flash")
  -> model.generate_content(my_message)
  -> print(response.text)
```

## Example Prompt

```python
my_message = "Write me three interesting facts about people."
```

## Expected Output

The notebook returns a natural-language answer generated by Gemini. The exact output changes from run to run because LLM responses are probabilistic.

## Notes

- The notebook currently uses the legacy `google-generativeai` package.
- Google may recommend the newer `google-genai` SDK for new projects, but this notebook is kept aligned with the code used in class.
- If the API key is missing or invalid, the Gemini request will fail during model generation.
